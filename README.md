# # Windows Server Active Directory Home Lab

## Project Overview

This project demonstrates a Windows Server Active Directory environment built using VirtualBox. The lab simulates a small business network and showcases common Help Desk and Systems Administration tasks.

## Environment

- Windows Server 2022
- Windows 11 Pro
- VirtualBox
- Active Directory Domain Services
- DNS

## Objectives

- Build an Active Directory domain
- Configure DNS
- Create Organizational Units
- Create and manage users
- Join Windows clients to the domain
- Practice common Help Desk tasks

## Network Configuration

- IP: 192.168.1.102
